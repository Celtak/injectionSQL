# Directions

## 1. Clone the repository

```bash
git clone https://github.com/Celtak/injectionSQL.git
```

## 2. Install the dependencies

```bash
composer require
```

## 3. Launch Docker

```bash
docker-compose up
```

## 4. Run the application

```bash
localhost:81
```

## 5. Run phpMyAdmin

```bash
localhost:82
```
